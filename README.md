# Prerequisite
Compose Version: 1.5.0

Kotlin Version: 1.9.0

Android Gradle Plugin Version: 8.2.0-beta01

Gradle: gradle-8.2-bin.zip

Java Version: 17

Android Studio Version: Hedgehog Beta 01

# Project Structure
```
├── data/
│   ├── datasource/
│   │   ├── local
│   │   └── remote
│   ├── di
│   ├── model
│   ├── repository
│   └── serializer
├── domain/
│   ├── repository
│   └── use_case
├── foundation/
│   ├── base
│   ├── extension
│   └── uicomponent

└── ui
```
